# 🗃️ PROJECT DATA LEDGER

## 1. 📌 Project Identity
- **Name**: BarberGo - Delivery Barber Service
- **Core Goal**: Platform connecting mobile barbers with clients at their homes. Real-time booking, GPS tracking, and secure payments.
- **Tech Stack**: 
  - Frontend: React + Vite + TypeScript + TailwindCSS
  - Backend: Node.js + Express.js + TypeScript
  - Database: PostgreSQL + Prisma ORM
  - Real-Time: Socket.io
  - Security: JWT + bcryptjs
  - Maps: Leaflet (open-source) or Google Maps API
- **Aesthetic**: Clean, professional, trustworthy. Dark mode option. Mobile-first design.

## 2. 🚦 Strict Rules
- Use TypeScript strictly (no `any` types)
- Implement proper error handling in all endpoints
- Follow RESTful API conventions
- Real-time updates for booking status
- GPS location validation for services
- Secure payment integration ready
